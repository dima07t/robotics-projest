# Project Description

This midterm robotics project focuses on SLAM-based mapping using a mobile robot in simulation.

A TurtleBot3 Burger robot was launched in a Gazebo virtual world using ROS 2 Jazzy. The robot was controlled manually with keyboard teleoperation. While moving through the simulated environment, Cartographer generated a 2D occupancy grid map in real time.

The generated map was visualized in RViz2 and then saved as output files for future use. The project demonstrates practical knowledge of robotics simulation, robot control, sensor-based mapping, and integration of ROS 2 packages in a complete SLAM pipeline.

Although the SLAM algorithm itself was not implemented from scratch, the project required full setup, configuration, execution, and testing of a working robotics mapping system.
